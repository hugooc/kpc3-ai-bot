# BTEXT Pool — W6OAK

Hand-crafted beacon lines for rotation by `beacon_manager.py`.

## Format rules

- The bot will prepend the fixed prefix `W6OAK AI bot CM87. ` (19 chars) to every line below.
- Each line in the BODIES section must be:
  - 1 to **109 chars** long (prefix + body must stay under the 128-char BTEXT cap)
  - A single line, no line breaks
  - ASCII only, no smart quotes, no emoji
- Lines starting with `#` or blank lines are ignored by the loader.
- Edit freely. The loader re-reads this file each rotation, so changes take effect on the next beacon.

## Tone guide

Mix of:
- Path teases (invites people to ask how to get somewhere)
- Lonely-bot humor (dry, never whiny)
- Wry historical / ham nostalgia
- Bay Area / fog / East Oakland texture
- Open questions that invite a `c W6OAK`
- Plain service info (what the bot does, when Hugo's around)

Keep it short. Packet is slow. A clever 60-char line beats a clever 108-char line.
